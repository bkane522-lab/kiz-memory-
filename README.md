# Kiz Memory

Kiz Memory est un prototype d’application qui transforme une soirée ou un workshop Kizomba en montage vertical prêt pour les réseaux sociaux.

## Fonctionnalités du prototype

- Accueil premium
- Import vidéo depuis la galerie
- Simulation d’analyse IA
- Détection simulée des meilleurs moments
- Format vertical 9:16
- Choix de durée : 15s, 30s, 45s, 60s
- Timeline magique
- Aperçu vertical
- Styles de montage
- Personnalisation titre / date / lieu
- Export social simulé
- Galerie souvenirs
- Memory Capsule
- Mode confidentialité
